## Единый формат

Каждый lab содержит:

- цель;
- симптом или задача;
- файлы для чтения;
- ограничения;
- ожидаемый результат;
- подсказки;
- эталонное рассуждение.

## Правило сложности

Если студент застрял:

1. сначала дать вопрос;
2. потом дать направление;
3. только потом дать файл;
4. строку кода давать последней.
