# Урок 2. Словарь Терминов

## Create Endpoint

Endpoint, который создает новый ресурс.

---

## POST

HTTP method, который обычно используется для отправки данных на сервер и создания нового ресурса или запуска действия.

---

## Request Body

Тело HTTP-запроса, в котором клиент отправляет данные серверу.

---

## `@RequestBody`

Аннотация Spring, которая связывает request body с параметром Java-метода.

---

## Request DTO

Java-объект, который описывает входной контракт API.

---

## Response DTO

Java-объект, который описывает выходной контракт API.

---

## `201 Created`

HTTP status code, который означает успешное создание ресурса.
