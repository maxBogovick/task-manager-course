# Урок 4. Справочник По Аннотациям И Понятиям

## `@PutMapping`

Связывает HTTP `PUT` запрос с методом controller.

---

## `@DeleteMapping`

Связывает HTTP `DELETE` запрос с методом controller.

---

## `ResponseEntity.noContent().build()`

Это способ вернуть HTTP `204 No Content`.

Он означает:

операция выполнена успешно, но body в ответе не нужен.

---

## Update Endpoint

Endpoint, который обновляет существующий ресурс.

---

## Delete Endpoint

Endpoint, который удаляет существующий ресурс.
