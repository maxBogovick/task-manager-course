# Урок 3. Словарь Терминов

## Path Variable

Часть path URL, которая идентифицирует конкретный ресурс.

---

## Query Parameter

Дополнительный параметр запроса, который уточняет выборку, фильтр, сортировку или пагинацию.

---

## Collection Endpoint

Endpoint, который возвращает набор ресурсов, а не один объект.

---

## Single Resource Endpoint

Endpoint, который возвращает один конкретный ресурс.

---

## Pagination

Механизм разбиения большого списка данных на страницы.

---

## `Pageable`

Spring abstraction для параметров страницы, размера и сортировки.

---

## `PagedResponse`

Модель ответа для списка/страницы ресурсов с метаданными.
