# Учебный Полигон

Учебные материалы организованы по этапам курса.

## Как устроена структура

Этап курса является основной единицей организации.

Внутри каждого `training/stages/stage-XX` лежат:

- `README.md` с быстрым входом;
- `LECTURE.md` с лекционным материалом;
- `TASK.md` с практическим заданием;
- `CHECKLIST.md` с критериями готовности;
- `start/` с starter-состоянием кода;
- `solution/` с эталонным решением, если оно уже подготовлено.
