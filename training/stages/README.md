## Как работать с этапом

1. Открыть `README.md`.
2. Прочитать `LECTURE.md`.
3. Выполнить `TASK.md`.
4. Проверить `CHECKLIST.md`.
5. Работать с кодом в `start/`.

Если есть `solution/`, его нужно смотреть только после самостоятельной попытки.
